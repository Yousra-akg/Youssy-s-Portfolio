Site de Location de Voitures

Ce projet est une plateforme moderne de location de voitures développée avec React et Redux. Il permet aux utilisateurs de rechercher, filtrer et réserver des véhicules facilement, avec une interface fluide et responsive. Les propriétaires peuvent également gérer leurs annonces en toute simplicité.

Technologies Utilisées

React Redux Tailwind CSS JavaScript (ES6) Node.js
